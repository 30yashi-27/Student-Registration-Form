# Student-Registration-Form
Created a Jawa Swing based application that allows the user to enter student details, validate inputs, and store them in a database

Description:
A GUI-based Java Swing application that allows users to register students by entering details such as Name, Age, Email, and Course.

Features:
- Input validation  
- Stores student details temporarily in an ArrayList  
- Displays registered students in a text area  

How to Run:
1. Save the file as `StudentRegistrationForm.java`
2. Compile and run:
   ```bash
   javac StudentRegistrationForm.java
   java StudentRegistrationForm
